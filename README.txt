Flor F — Landing Page (Netlify-ready)

Como usar:
1) Abre a pasta e substitui as imagens dentro de /assets/:
   - hero.jpg (imagem principal)
   - gallery1.jpg ... gallery6.jpg (galeria)
   Dica: usa JPGs otimizados (ideal 200–400KB cada).

2) Abre o index.html e ajusta texto / lista de flores se quiseres.

3) Deploy no Netlify:
   - Arrasta a pasta inteira (ou o ZIP) para o Netlify
   - Ou liga a um repositório GitHub

Notas:
- O formulário não envia email: ele abre o WhatsApp com a mensagem pré-preenchida.
- O Google Maps embed usa pesquisa por morada (sem API key).
